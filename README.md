# Clear 晨曦 | 理性的幸福主义者

> **Hacker. Maker. Thinker.**

这是一个基于 **Next.js 15** 构建的个人作品集与数字花园。它不仅展示了我的技术积累与产品设计，更是我作为一名“理性的幸福主义者”，在硬件探索、全栈开发与人文思考之间的交汇点。

## ✨ 特性

- **极简主义设计**：深度定制的 UI，追求极致的留白与排版美感。
- **MDX 内容系统**：支持高度自定义组件的 Markdown 系统，轻松管理项目（Works）与随笔（Blog）。
- **动效体验**：集成 Framer Motion 动画，包括 3D 悬浮明信片、磁性按钮及丝滑的页面转场。
- **响应式优化**：全设备适配，确保在手机、平板与桌面端都拥有完美的视觉呈现。
- **SEO & 品牌**：内置动态 OpenGraph 分享图生成，完善的元数据配置。

## 🛠️ 技术栈

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Content**: MDX + `next-mdx-remote`

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

## 📂 目录结构

- `src/app/`: 页面路由与布局配置
- `src/components/`: 可复用的 UI 组件与 MDX 渲染器
- `content/`: 存放作品与博客的 MDX 源文件
- `public/`: 静态资源文件
